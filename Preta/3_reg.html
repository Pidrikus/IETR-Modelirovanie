4. Регламентные работы

Назад к главному меню
Название работы Периодичность Описание работы
1. Разделение данных по отдельным точкам. Каждый раз при расширении области захвата территории приложением Консолидация данных многолетних наблюдений по темпами самовосстановления фитомассы отдельных точек и их разбиение по координатам.
2. Перевод координат Каждый раз при расширении области захвата территории приложением Перевод географических координат(B,L) в прямоугольные (x,y) методом референц-эллипсоида Красовского нажатием кнопки "Перевод координат" в приложении.
3. Задание F0 для необходимых фрагментов Во время каждого рабочего цикла Ввод начального количества фитомассы в таблицу для отдельных фрагментов дороги после вывода ее из эксплуатации.
4. Выбор временного периода и получение графиков экологического повреждение для выбранных фрагментов Во время каждого рабочего цикла 1) Ввод количества лет наблюдений необходимых для пользователя в текстовое поле "Права граница" 2) Построение графиков экологического повреждения D(T) для отдельных фрагментов автодороги и полностью для выбранной автодороги на кнопку "Построить графики".
